# MIDIGI Marketing Platform

Fullstack marketing platform built with Next.js, Supabase, Azure VPS and Cloudflare.

## Features

- Marketing landing page
- CRM dashboard
- Lead management
- Blog CMS
- Payment tracking
- Campaign analytics
- Authentication & role management
- Responsive admin dashboard

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- PM2
- Nginx
- Cloudflare
- Azure VPS
